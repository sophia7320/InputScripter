> 功能特性

按键连点

以指定间隔连续按下键盘按键

鼠标连点

以指定间隔连续点击鼠标按键

按键长按

模拟持续按住键盘按键

宏命令系统

自定义触发按键和模拟操作的宏

声音提示

操作开始和结束时提供蜂鸣提示

查看所有可用按键

列出支持的键盘和鼠标按键

> 系统要求

操作系统: Windows (主要支持)
Python版本: Python 3.6+
依赖库:
  - keyboard
  - mouse

> 命令行参数

参数 说明 兼容性
--key 按下键盘按键（可多次使用） 只能与--time连用
--mouse 点击鼠标按键（可多次使用） 只能与--time连用
--time 按键/鼠标点击间隔（秒） 只能与--key或--mouse连用
--press 持续按住键盘按键（可多次使用） 与其他参数冲突
--pmouse 持续按住鼠标按键（可多次使用） 与其他参数冲突
--view 查看可用按键列表 与其他参数冲突
--func 执行宏文件（func文件夹中的文件名） 与其他参数冲突
--showfunc 查看可用的宏文件列表 与其他参数冲突

> 使用示例

1. 按键连点

# 以0.5秒间隔连续按下a、b、c键
python InputScripter.py --key=a --key=b --key=c --time=0.5

2. 鼠标连点

# 以0.3秒间隔连续点击鼠标左键和右键
python InputScripter.py --mouse=left --mouse=right --time=0.3

3. 按键长按

# 持续按住按键2、3、6
python InputScripter.py --press=2 --press=3 --press=6

4. 执行宏命令

# 执行func文件夹中的macro.json宏文件
python InputScripter.py --func=macro

> 宏命令示例

[
  {
    "trigger": "F1",
    "options": [
      {"key": "1", "time": 0.1},
      {"key": "2", "time": 0.2},
      {"key": "3", "time": 0}
    ]
  },
  {
    "trigger": "F2",
    "options": [
      {"mouse": "left", "time": 0},
      {"mouse": "right", "time": 0}
    ]
  }
]

> 项目结构

InputScripter/ 
├── src/ 
│ ├── InputScripter.py # 主程序入口 
│ ├── func/ # 宏文件存储目录（运行时自动创建） 
│ ├── operation/ # 操作执行模块 
│ │ ├── click.py # 连点操作（键盘/鼠标） 
│ │ ├── press.py # 长按操作（键盘/鼠标） 
│ │ ├── view.py # 查看按键列表 
│ │ └── function.py # 宏命令执行 
│ └── song/ # 音效模块 
│ ├── beep.py # 蜂鸣提示音 
└── README.md # 项目说明文档
注意事项：
  • 某些功能可能需要以管理员权限运行
  • 所有持续操作都可以通过长按F8键停止
  • 操作启动后有2秒延迟，便于切换到目标窗口
  • 宏文件必须放在func文件夹中
  • 主要针对Windows系统设计，其他系统可能功能受限
  • 此类自动化工具在某些游戏中可能被视为作弊行为
